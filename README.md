# 📦 Inventory Management System

A full-stack web application for managing product inventory, built with **Python Flask** and **SQLite**.

## ✨ Features

- ✅ Add new products to inventory
- ✅ View all products in a clean table
- ✅ Edit product details (name, description, quantity, price)
- ✅ Delete products from inventory
- ✅ Search products by name
- ✅ Flash messages for user feedback

## 🛠️ Technologies Used

| Category | Technologies |
|----------|--------------|
| Backend | Python, Flask |
| Database | SQLite |
| Frontend | HTML, CSS |
| Templating | Jinja2 |

## 📋 How to Run Locally

```bash
# Clone the repository
git clone https://github.com/luwam-tesfamichael/inventory-management-system.git

# Navigate to the folder
cd inventory-management-system

# Install Flask
pip install flask

# Run the application
python app.py

# Open your browser to http://127.0.0.1:5000
